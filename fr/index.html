Présentation

Depuis maintenant 5 ans, les éléves de LLG organisent un tournoi de volley-ball interclasse. C'est un évènement très attendu par tous les participants, qui se déroule en 2 phases :

  • Une phase de poules où les classes d'un même niveau se rencontrent et à l'issue de laquelle les meilleures équipes sont qualifiées
  • Une phase éliminatoire où les classes qualifiées lors d'une ronde de 16 se rencontrent pour sacrer la classe championne

Cette année (2025-2026), le tournoi compte 33 équipes inscrites :
  • 7 (/7) classes de seconde
  • 7 (/9) classes de seconde
  • 8 (/9) classes de terminale
  • 11 classes de CPGE
